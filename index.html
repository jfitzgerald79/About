About: A Pathway to GIS Professionalism.

This site is dedicated to providing information in regards to my professional experience as a GIS specialist and soil technician. The following pages will take the user to interactive maps showing my resume in a spatial representation.

Code References, libraries, and GitHub

The interactive maps were created with HTML5, JavaScript, and CSS. More specifically, I used JQuery, Leaflet, geoJSON, and Bootstrap. Also, I used arcGIS API for JavaScript, GitHub, and GeoNet for additional guidance and references. Github was used to build this website, store data, and share data. It provides an appealing environment for novice programmers to easily use and implement HTML5 and JavaScript.

Interactive Maps

Click on the buttons called GIS Professional Experience and GIS Conferences. Each button will show a different aspect of my resume. The first interactive map will demonstrate education, employment, and internships. The second will show an interactive map of where I have attended conferences and where I have presented projects.

References Cited and Acknowledgements

Most of my references come from GitHub, Leaflet, ESRI's GeoNet, ESRI on GitHub, and StackExchange. Specific references include the Leaflet Tutorial, ESRI on GitHub, about legends, and ArcGIS API for Javascript.
